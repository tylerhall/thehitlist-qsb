thehistlist-qsb
===============

A [Google Quick Search Box](http://code.google.com/p/qsb-mac/) plugin to send items to [The Hit List](http://www.potionfactory.com/thehitlist/).

INSTALL
-------

 * Download the lastest copy of [the source](http://github.com/tylerhall/thehitlist-qsb/tree/master) using git or directly from GitHub.
 * Copy the plugin (the .hgs file) to `~/Application Support/Google/Quick Search Box/Plugins`

You're done!
